# Project Overview: AI Co-Founder Idea Validator

This directory contains the foundational documentation and planning materials for the **AI Co-Founder Idea Validator**, a web application designed to act as a strategic intelligence partner for entrepreneurs. The tool guides users through a 7-phase validation process, combining human input with AI-driven market research to stress-test business ideas.

## Directory Purpose
This is a **documentation and research repository** serving as the "Source of Truth" for the project's conceptual design, functional requirements, and technical architecture. It currently contains no source code but outlines the complete path for development.

## Key Files
- **`Idea Validator - PRD.md`**: The **final approved version** of the Product Requirements Document (v1.1). It is the primary source of truth, detailing the vision, user personas, 7-phase functional loop, system architecture, and data models.
- **`AI Coaching Idea Validation Flow .docx`**: Detailed interaction flow for the AI coaching engine.
- **`Valdating business ideas steps .docx`**: Procedural steps and methodology for the validation framework.
- **`Universal Project Blueprint (Checklist Template).docx`**: A standardized checklist for project execution and quality assurance.

## Planned Tech Stack
As defined in the PRD, the project is intended to be built using:
- **Framework**: Next.js 14 (App Router)
- **Database**: Neon Tech (Serverless PostgreSQL)
- **ORM**: Prisma
- **UI**: Shadcn/UI + Tailwind CSS
- **AI/LLM**: OpenAI GPT-4o
- **Search API**: Tavily API
- **Hosting**: Vercel
